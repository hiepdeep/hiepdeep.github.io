JavaScript Break and Continue
Câu lệnh break "nhảy ra" của một vòng lặp.
Trong ví dụ, câu lệnh break kết thúc vòng lặp ("ngắt" vòng lặp) khi bộ đếm vòng lặp (i) là 3.
							
								for (let i = 0; i < 10; i++) {
									if (i === 3) { break; }
									text += "The number is " + i + "<br>";
								}
								// 0 1 2
Câu lệnh continue "nhảy qua" một lần lặp trong vòng lặp.
Trong ví dụ, câu lệnh continue ngắt một lần lặp (trong vòng lặp), nếu một điều kiện xác định xảy ra và tiếp tục với lần lặp tiếp theo trong vòng lặp.
							
								for (let i = 0; i < 10; i++) {
									if (i === 3) { continue; }
									text += "The number is " + i + "<br>";
								}
								// 0 1 2 4 5 6 7 8 9
